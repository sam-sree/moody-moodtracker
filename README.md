#**MOODY.**


Mood Calendar

A simple, visual mood-tracking calendar built using vanilla HTML, CSS, and JavaScript.

The idea was to create something I would actually want to use daily, lightweight, private, and calm — instead of an overwhelming app with logins, notifications, or data tracking.

---

What it does

- Displays a monthly calendar view
- Click on any day to log:
  - your mood (using color-coded emojis)-upto 3 a day!
  - an optional note about the day
- Saves everything locally using `localStorage`
- Highlights:
  - days with notes
  - today’s date
- Lets you switch months without losing data

All data stays in your browser. Nothing is uploaded anywhere.

---

Why I built this

I wanted to practice:
- real JavaScript date logic (not hardcoded calendars)
- clean UI state handling
- localStorage-based persistence
- building something usable, not just “demo-looking”

This project helped me understand how small UX details, like fixed layouts and modal flows, affect how polished a product feels.

---

Tech stack

- HTML
- CSS (custom styling, no frameworks)
- Vanilla JavaScript
- Browser `localStorage`

No libraries, no frameworks, no backend.
